# gameboy-emulator

A Gameboy Emulator.

## Build and Run

Requirements:

- SDL2 is installed.
- Target system is little endian.

Run `make` to build the release version.

## Testing

Run tests with `test.sh`. Or run `make tests && ./run_tests`
